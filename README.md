## 구현 기능 목록
- [ ] 구입금액 입력요청 출력 
- [ ] 구입금액 입력
  - [x] 구입 금액이 정수인지 검증
  - [x] 구입 금액이 1,000의 배수인지 검증
- [ ] 구매한 로또 개수 출력
- [x] 로또 번호 생성
- [x] 로또 티켓 생성
- [x] 로또 티켓 저장
- [ ] 지난 주 당첨 번호 입력요청 출력
- [ ] 지난 주 당첨 번호 입력
- [ ] 당첨 번호는 `,`로 구분
- [ ] 당첨 번호 검사
  - [ ] 보너스 볼 번호가 정수인지 검사
  - [x] 당첨 번호가 6개인지 검사
  - [x] 당첨 번호가 45이하인지 검사 
- [ ] 보너스 볼 번호 입력 요청
- [ ] 보너스 볼 번호 입력
- [ ] 보너스 볼 번호 검사
  - [ ] 보너스 볼 번호가 정수인지 검사
  - [x] 보너스 볼 번호가 45이하인지 검사
- [ ] 당첨 번호와 구매한 로또 번호 비교
- [ ] 당첨 통계 출력
